4.1 Problem 1 - theoretical maximum channel capacity

Calculate the theoretical channel capacity using Shannon formula given the measurement results from the spectrum image. The instructions to solve the problem are in the lab guide section 2.

Given measurement results:

Center frequency shown on screen: 2.477 GHz

Span shown on screen: 100.0 MHz

Measured lowest (lower) frequency of used spectrum: 2.467 GHz

Measured highest (upper) frequency of used spectrum: 2.487 GHz

Measured width of used spectrum: 0.02 GHz

Signal power [dBm] (average of signal): -33.7 dB

Noise power [dBm] (average of noise level): -65 dB

Solution:

C = B⋅log2(1+S/N)

R = W*log2(1 + S/N)

B = 20MHz

S = -33.7 dB; P[mW] = 10^-33.7/10 = 0.0004266 mW

N = -65 dB; P[mW] = 10^-65/10 = 0.0000003162 mW

S/N[linear power ratio] = S[mW]/N[mW] = 0.0004266 / 0.0000003162 = 1349.146

с = 20 * log2(1+1349.146) = 208 Mbit/s
